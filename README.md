# KDShortner
A php code that helps you to Shorten link in a click in terminal
Commands for terminal
$ pkg install php
$ git clone https://github.com/KapoorDev/KDShortner
$ cd KDShortner
$ ls
$ php KDShortner.php
$ php KDShortner.php (link)

your link will be shortned
